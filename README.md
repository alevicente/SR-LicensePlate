# SR-LicensePlate
This repository stores the synthetic and real low-resolution images
containing license plate characters used in the reference:

Alexandre Natã Vicente, Helio Pedrini. "A Learning-Based Single-Image
Super-Resolution Method for Very Low Quality License Plate Images". IEEE
International Conference on Systems, Man, and Cybernetics (SMC), Budapest,
Hungary, October 09-12, 2016.
